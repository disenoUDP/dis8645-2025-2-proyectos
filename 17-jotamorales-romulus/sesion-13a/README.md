## Apuntes

Misaa mostró un referente visual para entender cómo funcionan los modelos generativos, sobre todo las GANs. En el video se veía cómo una imagen empieza como ruido total y, paso a paso, se va transformando hasta que aparecen formas que se reconocen. Lo importante era ver el proceso, no solo el resultado final. Se nota cómo la máquina va aprendiendo patrones y organizando la información.

- Todo empieza con ruido.

- La imagen se refina poco a poco.

- Las formas aparecen gradualmente.

- Lo clave es ver cómo se construye, no solo cómo queda al final.

- Se mencionó a Casey Reas, que muestra estas transformaciones de forma visual y artística.

- Esto ayuda a entender cómo se puede formar una imagen desde cero y cómo se organiza la información para que tenga sentido.

### Funcionamiento de la IA

La Inteligencia Artificial es un área de la informática que busca que las máquinas hagan tareas que normalmente haría una persona, como aprender, reconocer cosas o tomar decisiones. Funciona usando Machine Learning y Deep Learning, que analizan grandes cantidades de datos para encontrar patrones y poder predecir o generar cosas nuevas.

### Redes Neuronales

GANs:

- Generadora: crea datos nuevos (imágenes, texto).

- Discriminadora: revisa si son reales o no.

- Se entrenan juntas y mejoran con cada iteración.

CHAR-RNN:

- Trabaja letra por letra.

- Aprende patrones y puede generar texto nuevo siguiendo el estilo del original.

### Matbutom – Tipografía Generativa

[Proyecto Tipográfica Propagandística](https://github.com/matbutom/typo-grafica-propagandistica)

Mateo está haciendo un proyecto para que la computadora pueda crear nuevas letras y afiches políticos a partir de ejemplos reales.

Cómo funciona:

- El programa reconoce las palabras en imágenes o textos.

- Luego separa cada palabra del resto.

- Después recorta las letras una por una.
