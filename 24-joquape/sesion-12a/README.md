## Clase Misa
Hablamos sobre qué es un objeto → cualquier cosa puede serlo: cosas físicas, lugares, pensamientos, imágenes mentales, etc.

También vimos metáforas → se forman mezclando cualidades no esenciales de objetos que no tienen relación. Ejemplo de clase: completo italiano + reproductor de música.

A partir de esto entramos al concepto de máquinas sentimentales, que son objetos o dispositivos diseñados para generar emociones específicas y raras, no genéricas (miedo a una hora exacta, procesos lentos para cosas rápidas, experiencias absurdas, etc.).

### Máquinas sentimentales que elegimos en el grupo

Después de compartir ideas entre todas, nos quedamos con estas 3 como las más interesantes y con mayor potencial:

Máquina generadora de estrés
→ Una máquina que mediante alarmas constantes te recuerda todo lo que deberías estar haciendo, aunque no quieras escucharlo.

Máquina dependiente de música
→ Una mascota virtual que literalmente vive de escuchar música; si no le produces música, se apaga / muere rápidamente.

Máquina selectora de importancia
→ Una máquina que cada día decide por su cuenta cuáles alarmas de tu teléfono son importantes y cuáles no.
