Mi idea proyecto:
Se trata de una máquina compuesta por tres botones y una pantalla.
El primer botón sirve para borrar un carácter, el segundo, ubicado al centro, permite recorrer todo el abecedario, una letra a la vez, como en los antiguos celulares donde había que pulsar repetidamente para elegir cada letra. Finalmente, el botón de la derecha funciona como espacio entre palabras.

Esta máquina busca provocar una sensación de frustración y nostalgia: frustración por la lentitud y dificultad del proceso de escritura, y nostalgia por evocar la estética y el funcionamiento de las tecnologías retro. 

## Clase
Conceptos vistos
Inteligencia Artificial y aprendizaje automático

- La IA funciona procesando grandes cantidades de datos para identificar patrones y tomar decisiones sin programación explícita.
- El aprendizaje automático permite mejorar con más datos; el aprendizaje profundo usa redes neuronales inspiradas en el cerebro.
- Las imágenes y palabras se representan como matrices de números. La IA aprende modificando estos valores según los ejemplos que recibe.
- Una época es una pasada completa por todos los datos de entrenamiento. Para que la IA aprenda bien se requieren miles de épocas.
- El espacio latente es una representación matemática simplificada donde datos similares quedan agrupados, permitiendo generar variaciones de imágenes y texto.
- Stable Diffusion es un modelo generativo capaz de crear imágenes realistas a partir de texto.

### Librerías, herramientas y servicios

- ml5.js: acerca el aprendizaje automático a artistas y estudiantes, basado en TensorFlow.js, usable directamente en el navegador.
- Google Colab: servicio en la nube para ejecutar Python, con acceso a GPU y TPU sin instalar nada en el computador.
- Captcha: sistema para distinguir humanos de bots mediante desafío-respuesta.

### Hardware para IA

- CPU: realiza todo tipo de tareas, pero más lento para IA.
- GPU: excelente para cálculos en paralelo, ideal para redes neuronales.
- TPU: hardware especializado de Google para acelerar IA (algunos Mac actuales incluyen una).

### Tecnología y empresas

- Qualcomm: empresa de semiconductores y tecnologías inalámbricas.
- Qualcomm adquirió Arduino para integrar tecnologías como IA manteniendo el enfoque open-source y la comunidad.
- El mil: unidad de medida anglosajona equivalente a 0,0254 mm.

## Ideas para Proyecto N°3

Concepto general:
Una máquina que reaccione cuando se le aplica fuerza.

Preguntas clave:

- ¿Qué pasa cuando recibe mucha fuerza?
- ¿Qué pasa cuando recibe poca?

Ideas exploradas:

- Punching bag que se calma al recibir un golpe.
- Uso de sensores FSR para medir presión (sensibles pero poco precisos).
- Fluidos no newtonianos y reacciones de materiales ante presión.
- Medición del impacto o velocidad de la fuerza aplicada.
- Incorporar temas contingentes o sociales en la experiencia.
- Investigar materialidades ligadas a la naturaleza (líquidos, materia inestable, etc.).
