## ¿Qué haremos para el Proyecto 03?

### Exploración inicial
Durante la sesión se discutió libremente sobre posibles direcciones para el proyecto.  
Se propuso abrir el proceso a ideas experimentales y a la reinvención de objetos cotidianos desde nuevas perspectivas.

### Lluvia de ideas
- Panel LED interactivo  
- Mascota virtual (tipo Tamagotchi)  
- Objetos cotidianos reinterpretados desde un enfoque emocional  
- Máquinas con características o comportamientos humanos

### Ideas más específicas
- **Panel LED emocional:** reacciona a la proximidad o al ruido con patrones lumínicos que reflejan “estados de ánimo”.  
- **Mascota virtual física:** comportamiento expresivo mediante motores, luces y movimientos simples (similar a un Tamagotchi materializado).  
- **Objetos con comportamientos animales o instintivos:** tiemblan, respiran, se esconden, reaccionan según estímulos.
