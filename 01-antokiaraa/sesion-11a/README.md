# sesion-11a

## Qué haremos para el proyecto 03?

- Pensar y proponer ideas libremente
- Lluvia de ideas: panel LED interactivo, mascota virtual (tamagochi)
- Reinventar objetos desde otro enfoque
- Máquinas con características humanas

### Más específicas

- Panel LED interactivo que responde emocionalmente a proximidad o ruido.
- Mascota virtual (tipo Tamagotchi), con comportamientos expresivos en base a motores y luces.
- Objetos que reaccionan como animales: tiembla, respira, se ocultan, etc.
