>Luego nos explicaron y reiteraron el uso de las metáforas

Metáfora por interacción método de Harman

Consiste en elegir dos objetos muy distintos, identificar sus cualidades y luego imaginar qué pasa cuando se encuentran. De esa interacción nace una imagen nueva: una metáfora.

Ejemplo:

- Algodón + fuego -> el algodón se quema -> Algodón en llamas.

Sirve para generar ideas en diseño, arte y escritura de forma rápida y conceptual.

>Manifiesto: Un manifiesto es una declaración escrita que expone las ideas, intenciones, principios y propósitos de una persona, grupo o movimiento, con el fin de inspirar un cambio o persuadir a otros.
