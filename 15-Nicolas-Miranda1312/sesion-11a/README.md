##### HDI

significa Human Interface Device (en español, Dispositivo de Interfaz Humana).

Es un tipo de dispositivo que permite la interacción entre una persona y una computadora u otro sistema electrónico. Los HIDs pueden recibir información del usuario (como un teclado, mouse o joystick) o entregar información al usuario (como una pantalla, un control con vibración o luces).

En resumen, un HID es cualquier dispositivo diseñado para comunicar al ser humano con una máquina de forma directa e intuitiva.
