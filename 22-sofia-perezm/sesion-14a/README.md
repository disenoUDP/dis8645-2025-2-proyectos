## Trabajo autónomo 
### Investigación de arduino uno r4 wifi

| Parte                                                        | Función                                                                                                                            |
| ------------------------------------------------------------ | ---------------------------------------------------------------------------------------------------------------------------------- |
| **Microcontrolador (Renesas RA4M1, ARM Cortex-M4 a 48 MHz)** | Es el “cerebro” del Arduino. Ejecuta el código que escribes y controla todo lo que ocurre en la placa.                             |
| **Memoria Flash (256 KB)**                                   | Donde se guarda tu programa (el sketch).                                                                                           |
| **RAM (32 KB)**                                              | Espacio para las variables y procesos mientras el programa corre.                                                                  |
| **WiFi y Bluetooth (módulo ESP32-S3)**                       | Permite conectar el Arduino a internet, redes locales o a otros dispositivos inalámbricos. Puedes enviar/recibir datos sin cables. |
| **Puertos de entrada/salida (E/S)**                          | Pines donde conectas sensores, motores, luces, etc. Hay digitales y analógicos.                                                    |
| **Conversor ADC (12 bits)**                                  | Convierte señales analógicas (como un potenciómetro o sensor de luz) en valores digitales que el microcontrolador puede entender.  |
| **Conector USB-C**                                           | Sirve para alimentar la placa y para cargar programas desde tu computadora.                                                        |
| **Alimentación**                                             | Puede funcionar conectada por USB o con una fuente externa (hasta 24V).                                                            |
| **LED matriz integrada (12x8)**                              | Una pequeña pantalla LED integrada que puedes programar para mostrar gráficos, texto o animaciones.                                |
| **Botones y LEDs integrados**                                | Incluye un botón de “reset” y algunos LEDs de estado (por ejemplo, para WiFi o alimentación).                                      |

### Sinestesia 
La sinestesia es un fenómeno perceptual en el que la estimulación de una vía sensorial o cognitiva provoca experiencias involuntarias en otra vía, como ver colores al escuchar música o sentir sabores al oler algo. Es como tener los sentidos "cruzados" y puede variar de persona a persona, siendo un rasgo que algunos artistas han utilizado en sus obras.
